Controls:
<left> and <right> arrow keys to move paddle
<space> to launch ball

Optionally:
Mouse to move paddle
<left mouse button> to launch ball


Features:
Angled blocks, ball acts according to angle
Blocks can take more than one hit to destroy
Time slows down if you are about to lose a life
mouse and keyboard can be used interchangably

Asthetics:
Sparks
Adjecent blocks light up like a wave when a block is hit
Ball changes size and color when it hits something
Ball has motion blur
